A trouver un événement

  1. Je rentre le nom de ma ville
  2. Je choisis les domaines recherchés
  3. Je consulte les événements près de chez moi
  4. Je souhaite en savoir plus sur un événement



A diffuser un événement

Je suis un catho isolé

  1. Je rentre mon nom, mon email et mon pseudo
  2. Je choisis les mots clefs
  3. Je rentre la description de l'événement

Je represente une paroisse ou une association

  1. Je rentre le nom de la paroisse ou de l'association
  2. Je choisis les mots clefs
  3. Je rentre la description de l'événement
